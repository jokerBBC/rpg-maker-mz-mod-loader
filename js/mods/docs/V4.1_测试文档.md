# ModLoader V4.1 测试文档

> **适用版本**：V4.1.0 / V4.1.1  
> **关联**：[开发规范](RMMZ_ModLoader_开发规范.md) · [实施计划](V4.1_unified_package_plan.md) · [使用手册](使用手册.md)

本文件为 V4.1 统一包结构后的 **完整手工测试清单**，不再区分「迁移前/迁移后」。作者自用文件迁移脚本见实施计划 §6；玩家侧 `mod_config` 旧键兼容见 V4.1.1（保存一次自动升级）。

---

## 测试前确认

- [ ] 管理器版本：**V4.1.1**（或当前发布版）
- [ ] `modloader_config.json` → `workshop.enabled: true`（测工坊时）
- [ ] Steam 自测包已 flatten（`<fileId>/` 包根直接放 `.js`，无 `js/mods/` 子目录）
- [ ] 本地 Mod 位于 `_localmods/<包名>/`，根目录 `js/mods/*.js`（除 ModLoader.js）**不应被扫描**
- [ ] 测试用本地包与真实 Mod 分开；删除测试勿动生产包

**结果记录**：每项 `[ ]` 后填 `✅` / `❌` / `⏭` 及备注。
